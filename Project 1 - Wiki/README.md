1. Download the project :

```
https://raw.githubusercontent.com/aBrihoum/CS50W/master/zipped-src/project-1-src.zip
```

2. Navigate to `Project 1 - Wiki/`.

3. Run `python manage.py runserver`.

4. Open http://127.0.0.1:8000/.
