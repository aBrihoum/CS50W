```
https://downgit.github.io/#/home?url=https://github.com/aBrihoum/CS50W/tree/master/Project%201/wiki
```

2. Navigate to `wiki/`.

3. Run `python manage.py runserver`.

4. Open http://127.0.0.1:8000/.
